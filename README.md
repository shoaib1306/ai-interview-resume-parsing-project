
# AI-Powered Interview Preparation & ATS Resume Analyzer

An intelligent full-stack web application that helps users optimize resumes, analyze job descriptions, and prepare for interviews using AI.

---

## 🚀 Features

- 📄 Resume PDF Upload & Parsing  
- 🎯 Resume vs Job Description Match Score  
- 🤖 AI-Based Technical Interview Questions  
- 💬 Behavioral Interview Questions & Answers  
- 📊 Skill Gap Analysis  
- 🛠 Personalized Preparation Roadmap  
- 📑 ATS-Friendly Resume Generation  
- 🔐 Secure User Authentication  
- 💾 Save Previous Reports & Resume History  

---

### Frontend

- Google Gemini API

---

## 📌 How It Works

1. User uploads resume PDF  
2. System extracts text from PDF  
3. User enters job description  
4. AI compares resume with job role  
5. Generates:
   - Match Percentage
   - Missing Skills
   - Interview Questions
   - Suggested Answers
   - ATS Resume
   - Roadmap to Improve
